# upzon
On-Page SEO Crawler Tool with Interface

Requirements

*Mysql Connector
*URLlib
*Requests
*BeautifulSoup
*Json

Places to edit:
* ROBOTS.TXT URL on upzon.py
* Sitemap URL on upzon.py
* upzon.py DB Connection Section
* config/config.php DB Connection for Interface
